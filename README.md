# SoftwareSystemOfDlpu
****大学软件应用研究室系统

## 实现功能
- 研究室成员管理
- 相册功能
- 新闻发布
- 成果展示
- 简易论坛
- 在线答题

## 主要技术
Jsp + Servlet + MVC + JDBC + Mysql

## 项目管理
Maven

## IDE
Intellij IDEA

## 项目打包
1. git clone 项目到本地
2. 切到该项目目录, 使用 mvn clean package打包

## 项目运行
1. 解压war包
2. 将其拷贝到Tomccat服务器下的webapps目录下
3. 然后启动tomcat服务器
4. 在浏览器中输入: http://localhost:8080/站点名，即可访问(8080为默认的Tomcat端口)
